Portfólio

Sistema Robótico Automatizado de Inspeção por Visão

Um sistema robótico automatizado projetado para análise abrangente de inspeção visual. O conceito oferece uma plataforma flexível capaz de inspecionar qualquer tipo de objeto ou componente. Este sistema foi desenvolvido para inspecionar mais de 200 pontos de inspeção distintos em um bloco de motor, concluindo todo o processo em apenas 90 segundos.

Tecnologias: Câmeras Cognex, Machine Learning (TensorFlow), Oracle, Python, HTML, CSS, JavaScript
